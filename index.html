多人协作数据更新指南

📋 方案一：腾讯文档在线协作（推荐）
适用场景：团队成员共同编辑数据，实时协作
操作步骤：
  1. 在腾讯文档创建在线表格（或用现有Excel导入）
  2. 按模板格式填写客户数据（点击下方"下载模板"获取格式）
  3. 将表格分享给团队成员，设置"可编辑"权限
  4. 数据更新后，在腾讯文档中点击「文件 → 导出 → CSV」
  5. 将导出的 CSV 文件拖拽到看板页面或点击"导入数据"上传
💡 提示：也可以把腾讯文档的共享链接发给有 Python 环境的人，运行 sync_data.py 脚本自动同步到云端，这样所有看板用户都能看到最新数据。
📁 方案二：直接拖拽上传
适用场景：快速更新，单人操作
  1. 下载模板，编辑客户数据
  2. 将 Excel/CSV 文件直接拖拽到看板页面任意位置
  3. 数据即时更新到当前浏览器
⚠️ 此方式仅更新本地数据，不会同步给其他人。如需共享，请用方案一 + sync_data.py。
🔄 方案三：脚本自动同步（需要 Python 环境）
适用场景：定时同步，数据发布到云端供所有人使用
python3 sync_data.py ~/Downloads/运营数据.xlsx
运行后将自动生成 data.json 并推送到 GitHub，看板页面刷新即可看到最新数据。
